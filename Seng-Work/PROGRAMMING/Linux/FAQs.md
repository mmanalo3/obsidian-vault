# How to create VNC server session?
- [i] execute `vncserver :30 -geometry 1920x1080 -SecurityTypes None -depth 8 -cc 4 -pixelformat RGB888`. Use `-geometry 2560x1440` for larger display

# How to tar directory?
- [i] execute `tar -cvf /path/to/destination/archive-name.tar /path/to/directory/to/tar` to tar a directory. Add `-z` for gzip.
