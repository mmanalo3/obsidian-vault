# How to create a submodule with a git repository?
- [i] execute `git submodule add <submodule_url> path/to/directory` where `path/to/directory` is not existing

# How to generate ssh-keys?
- [i] execute `ssh-keygen -t ed25519 -C "email@example.com"`. If unknown key type ed25519 is encountered use `ssh-keygen -t rsa -b 4096 -C "email@example.com` instead. #ssh
