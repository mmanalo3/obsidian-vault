# Logic state characters
| **Character** | **Direction** | **Logic State (Tester Perspective)** |
| ------------- | ------------- | ------------------------------------ |
| L             | Input         | Drive low                            |
| H             | Input         | Drive high                           |
| C             | Input         | Drive pulse                          |
| Y             | Input         | Don't care                           |
| 0             | Output        | Expect low                           |
| 1             | Output        | Expect high                          |
| Z             | Output        | Expect high-impedance state          |
| M             | Output        | Mask (don't care)                    |
